# Minecraft 1.20.x (1.20.5-1.20.6) Player-vs-Player Datapack and Resource pack
[English](../README.md) / 日本語  
このリポジトリはMinecraft 1.20.x(特に、1.20.5以降)向けに開発されたデータパックとリソースパックを内包します。
